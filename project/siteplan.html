PROJECT SITE PLAN

Site Name

Site Name: GCASH Transaction Tracker

Reason for Selection: The name directly reflects the website’s primary function—tracking GCASH transactions. It’s clear, easy to understand, and immediately tells users what the site is about. Since it’s intended for internal business use, the straightforward naming makes it ideal for daily reference.

Site Purpose

This website is built to help me manage and track the cash flow of my business by logging GCASH transactions such as Cash In and Cash Out. Instead of using a traditional database, the site uses Sheet.best to connect to a Google Sheet for storing and retrieving transaction data via API. This makes the system lightweight, cost-effective, and easy to maintain, especially for solo or small-scale business operations. The dashboard provides an overview of total cash in, cash out, and the current balance to support simple financial monitoring.

Scenarios

  1. How can I log a new GCASH transaction and specify whether it is a Cash In or Cash Out?
  2. What is the current balance based on all transactions recorded this month?
  3. How much total Cash In and Cash Out has been recorded over time?
  4. Can I review recent transactions to reconcile with my GCASH wallet?

Color Schema

Body Background Color: #c5c5c5 (Light Gray)

Usage: Provides a clean and neutral background that keeps the interface light, readable, and business-like.

Main Accent Color: #2a6edc (Strong Blue)

Usage: This color will be applied to sections like the header, footer, and content blocks, creating a contrasting and sophisticated look against the body background.

Card & Form Background: #ffffff (White)

Usage: Keeps forms and transaction cards clear and focused, improving readability of numbers and user inputs.

Button Hover Color:#1d54b4 (Darker Blue)

Usage: Enhances interactivity and visual feedback on buttons during user action.

Typography

Title Font: Righteous

Usage: This font will be used for all titles (h1, h2, h3) to create a distinctive and bold visual impact that enhances the overall aesthetic of the website.

Body Text Font: Roboto

Usage: This font will be used for the main body text to ensure readability and a clean, modern appearance throughout the site.
